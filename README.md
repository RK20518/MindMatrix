# MindMatrix

## Overview

MindMatrix is an AI-powered mental health analysis application that uses Retrieval-Augmented Generation (RAG) and Large Language Models (LLMs) to identify possible mental health conditions from user text. The system retrieves relevant information from a mental health dataset and generates explainable responses.

## Features

* Mental health text analysis
* Retrieval-Augmented Generation (RAG)
* LLM-based response generation
* Streamlit web interface
* Semantic embeddings for information retrieval
* Explainable AI responses

## Technologies Used

* Python
* Streamlit
* ChromaDB
* Sentence Transformers
* Groq API
* Llama
* Pandas

## Installation

1. Clone the repository:

```bash
git clone https://github.com/RK20518/MindMatrix.git
```

2. Navigate to the project folder:

```bash
cd MindMatrix
```

3. Install the required packages:

```bash
pip install -r requirements.txt
```

4. Run the application:

```bash
streamlit run app.py
```
